\# Fase 05 — Implantação da aplicação HumanGov utilizando Amazon EKS



Nesta etapa do projeto foi realizada a implantação da aplicação HumanGov utilizando Kubernetes no Amazon Elastic Kubernetes Service (EKS).



Durante esta fase foram abordados os seguintes componentes da arquitetura:



\- Configuração da Amazon VPC

\- Configuração do cluster Amazon EKS

\- Utilização do kubectl

\- Utilização do eksctl

\- Construção das imagens Docker

\- Publicação das imagens no Amazon ECR

\- Criação dos manifestos Kubernetes

\- Configuração do Amazon Route 53

\- Configuração do Application Load Balancer (ALB)

\- Configuração do Ingress

\- Utilização do AWS Certificate Manager (ACM)



\## Estrutura criada



Nesta fase foram adicionados os manifestos Kubernetes utilizados para representar a implantação da aplicação HumanGov.
