\# Cluster Amazon EKS



Durante esta fase foi realizada a preparação do ambiente Kubernetes na AWS.



Os principais componentes utilizados foram:



\- Amazon VPC

\- Amazon EKS

\- kubectl

\- eksctl



Após a criação do cluster, a aplicação HumanGov foi preparada para implantação utilizando manifestos Kubernetes.



Esta documentação representa a configuração do ambiente descrita na Fase 5 do projeto HumanGov.
