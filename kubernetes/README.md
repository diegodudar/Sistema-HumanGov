\# Kubernetes



\## Fase 5 – Implantação da aplicação HumanGov na AWS EKS



Nesta fase do projeto HumanGov foi realizada a implantação da aplicação utilizando Amazon Elastic Kubernetes Service (EKS).



A aplicação foi implantada em um cluster Kubernetes utilizando imagens Docker armazenadas no Amazon Elastic Container Registry (ECR).



Também foram utilizados recursos de rede e segurança da AWS para disponibilização da aplicação.



\## Componentes utilizados



\- Amazon EKS

\- Kubernetes

\- Docker

\- Amazon ECR

\- Amazon Route 53

\- AWS Certificate Manager (ACM)

\- Application Load Balancer (ALB)



Esta implementação representa a Fase 5 do projeto HumanGov.
