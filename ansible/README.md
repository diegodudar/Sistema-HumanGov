# Ansible

Nesta fase foi utilizada a ferramenta Ansible para automatizar a configuração das instâncias Amazon EC2 e realizar a implantação da aplicação HumanGov.

O provisionamento da infraestrutura permanece sob responsabilidade do Terraform, enquanto o Ansible é utilizado para configuração e deploy da aplicação.

Arquivos principais:

- inventory
- playbook
- documentação
