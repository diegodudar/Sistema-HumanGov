\# Fase 04 — Conteinerização e preparação da aplicação para implantação na AWS



Nesta etapa do projeto HumanGov foi iniciado o processo de implantação da aplicação em ambiente conteinerizado.



A aplicação HumanGov foi empacotada utilizando Docker, juntamente com uma imagem NGINX utilizada como front-end da solução.



As imagens produzidas nesta fase são preparadas para armazenamento no Amazon Elastic Container Registry (ECR) e posterior utilização pelo Amazon Elastic Container Service (ECS).



\## Tecnologias utilizadas



\- Docker

\- Python

\- NGINX

\- Amazon ECR

\- Amazon ECS

\- AWS Fargate



\## Entregas da fase



\- Dockerfile da aplicação HumanGov

\- Dockerfile do NGINX

\- Configuração inicial do NGINX

\- Estrutura para conteinerização da aplicação



Esta implementação corresponde à Fase 4 do projeto HumanGov.
