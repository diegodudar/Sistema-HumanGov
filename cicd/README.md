\# CI/CD



\## Fase 6 – Automação da implantação da aplicação HumanGov



Nesta fase do projeto HumanGov foi implementado o processo de Integração Contínua (CI) e Entrega Contínua (CD) utilizando serviços da AWS.



A automação foi baseada na integração entre o AWS CodeCommit, AWS CodeBuild e AWS CodePipeline, permitindo que alterações realizadas no código-fonte fossem processadas automaticamente até a implantação da aplicação.



\## Componentes utilizados



\- AWS CodeCommit

\- AWS CodeBuild

\- AWS CodePipeline

\- Docker

\- Amazon Elastic Container Registry (ECR)

\- Kubernetes

\- Amazon Elastic Kubernetes Service (EKS)
