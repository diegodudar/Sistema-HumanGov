\# Fase 06 — Automação da implantação utilizando AWS CI/CD



Nesta etapa do projeto HumanGov foi automatizado o processo de construção e implantação da aplicação utilizando os serviços de CI/CD da AWS.



Durante esta fase foram abordados os seguintes componentes:



\- AWS CodeCommit

\- AWS CodeBuild

\- AWS CodePipeline

\- Docker

\- Amazon Elastic Container Registry (ECR)

\- Kubernetes

\- Amazon Elastic Kubernetes Service (EKS)



O pipeline conecta o repositório de código-fonte ao processo de construção e implantação da aplicação, permitindo a automação do fluxo de entrega.



Esta documentação representa a Fase 6 do projeto HumanGov.
