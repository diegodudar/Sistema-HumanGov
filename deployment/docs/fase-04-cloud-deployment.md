\# Cloud Deployment



Após a conteinerização da aplicação HumanGov, foi iniciado o processo de implantação em ambiente AWS.



Durante esta etapa foram realizadas as seguintes atividades:



\- Construção das imagens Docker.

\- Publicação das imagens no Amazon Elastic Container Registry (ECR).

\- Criação do cluster Amazon ECS.

\- Configuração da Task Definition.

\- Implantação da aplicação utilizando AWS Fargate.

\- Integração da aplicação através do Application Load Balancer (ALB).



Esta etapa representa a evolução da aplicação HumanGov para um ambiente conteinerizado executado na AWS.



\## Ambiente utilizado



Durante esta etapa do projeto foram utilizados os seguintes recursos da AWS para preparação do ambiente de implantação:



\- AWS Cloud9

\- AWS IAM

\- Amazon Elastic Container Registry (ECR)

\- Amazon Elastic Container Service (ECS)

\- AWS Fargate

\- Application Load Balancer (ALB)



Esses componentes foram utilizados durante o processo de conteinerização e implantação da aplicação HumanGov na AWS.
